# QuickWash AI

An AI-powered Laundry Order and Pickup Management System built with Laravel.

## Features

- User Registration & Login
- Online Laundry Booking
- Pickup & Delivery Scheduling
- Order Tracking
- AI-Based Service Recommendation
- Admin Dashboard

## Tech Stack

- Laravel 12
- PHP 8
- MySQL
- HTML/CSS
- JavaScript
- Vite
